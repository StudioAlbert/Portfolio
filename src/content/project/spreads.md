Installation vidéo expérimentant les nouveaux terrains de jeu du vjaying : intervention de sources lumineuses physiques mêlées à la vidéo, jeu sur la matière imprimée et les techniques de projection.

## Tech

- Arduino + LEDs pour les sources lumineuses synchronisées
- Pipeline vidéo / VJ
- Combinaison impression et techniques projectives
