Support de cours sur l'**Utility AI** : une approche de décision où chaque action reçoit un score continu calculé via une **courbe de réponse** (linéaire, quadratique, logistique…). Le NPC choisit ensuite l'action au meilleur score.

## À tester

- Ajuste les paramètres des courbes et observe l'effet sur le score.
- Compare les familles de courbes pour comprendre quand utiliser chacune.

La démo jouable est embarquée plus bas (champ `demoUrl`). Code source : voir le lien repo.
